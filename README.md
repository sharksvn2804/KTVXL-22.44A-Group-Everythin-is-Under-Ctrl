(Lý do: project này được sửa đổi từ example Temperature Sensor của ESP RainMaker)

Mở thư mục esp-rainmaker/examples/ trong bộ SDK.

Copy thư mục vừa đổi tên vào đây, chọn Replace khi được hỏi.
